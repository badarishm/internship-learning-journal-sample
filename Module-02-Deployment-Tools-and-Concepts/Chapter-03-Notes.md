Cloud Development & Automation:
This session covered cloud-based development using GitHub Codespaces, deploying applications on Hugging Face Spaces, and automating workflows with GitHub Actions.

Key Learnings:

1.Git, WSL & VS Code (Q&A):
- Clarified common setup issues with Git, WSL, and VS Code.
- Understood importance of correct environment configuration.

2.GitHub Codespaces:
- Learned how to use **GitHub Codespaces** for cloud-based development.
- Codespaces provides a ready-to-use development environment in the browser.
- No need for heavy local setup.

3.Hugging Face Spaces:
- Deployed applications using **Hugging Face Spaces**.
- Understood how to host ML/AI apps in the cloud.
- Learned basics of deploying apps with public access.

4.GitHub Actions:
- Introduction to **GitHub Actions** for automation.
- Automates tasks like testing, building, and deployment.
- Helps create CI/CD pipelines for projects.

What I Took Away:
- Cloud environments simplify development and collaboration.
- Hugging Face Spaces is useful for deploying AI applications.
- GitHub Actions helps automate workflows and improve efficiency.
- Automation and cloud deployment are important industry skills.
