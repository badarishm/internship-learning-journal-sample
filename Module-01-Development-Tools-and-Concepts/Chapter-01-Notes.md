In this session, we were introduced to essential development tools required for the course. Setting up the correct development environment is important before starting practical work.

WSL:
WSL (Windows Subsystem for Linux) allows users to run a Linux environment directly inside Windows without using a virtual machine.

It provides:
-Linux terminal access
-Compatibility with development tools
-Lightweight system integration

Importance of Linux for developers:
-Many production servers run on Linux.
-Better compatibility with open-source tools.
-Efficient package management using apt.
-Strong command-line support.

Ubuntu 24.04 LTS:
Ubuntu is a popular Linux distribution. LTS (Long Term Support) version ensures stability and long-term updates.

In this session, Ubuntu 24.04 was chosen for installation.

WSL Commands Explained:
1.wsl --update
Updates the Windows Subsystem for Linux to the latest version.

2.wsl --list --online
Displays available Linux distributions that can be installed.

3.wsl --install Ubuntu-24.04
Installs Ubuntu 24.04 distribution.

4.sudo apt update && sudo apt upgrade -y
Updates and upgrades installed packages inside Ubuntu.

Importance of Environment Setup:
Before learning advanced tools, it is necessary to:

-Configure development environment
-Install required tools
-Ensure system compatibility
-Proper setup avoids future technical issues.
